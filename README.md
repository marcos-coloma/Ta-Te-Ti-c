# Ta-Te-Ti en C

Este es un proyecto de aprendizaje hecho en C

El tateti tiene un tablero dinamico (de 3x3 a 7x7), se puede jugar:
- jugador vs jugador
- jugador vs CPU

La CPU tiene 2 dificultades:
- Modo normal: ejecuta movimientos aleatorios
- Modo dificil: juega de forma mas inteligente, priorizando bloquear jugadas ganadoras.

Para compilar el proyecto:
- mingw32-make

Para ejecutar la aplicación:
- ./build/db_test

nota: si no podes compilar o ejecutar fijate el makefile
